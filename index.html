What is NeptunePowered? NeptunePowered is a team dedicated to making server owners lives easier. What is Neptune? Neptune is a work-in-progress implementation of CanaryLib for ‘Vanilla’ Minecraft. It will be implemented using Mixin. Neptune will also use mcp mappings. Why? I am a big supporter of Canary, and feel (in comparison to other server mods) Canary lacks a great implementation.
